🔍 How to Use the Bug Predictor

  1. Go to the PREDICTOR tab
  2. Select your code language (C or Java)
  3. Enter the filename
  4. Paste your code snippet
  5. Click ANALYZE CODE
  6. View the bug detection result and confidence score

🐛 Supported Bug Types

  • Null Pointer Exception: Dereferencing NULL pointers
  • Buffer Overflow: Writing beyond array bounds
  • Segmentation Fault: Invalid memory access
  • Use After Free: Using freed memory
  • Out of Bounds: Array index out of range
  • Memory Leak: Unreleased allocated memory
  • Race Condition: Multi-threading conflicts
